我想自己搭一個木製陽台：Deck Building 101

by Matt · 約 5 分鐘讀完

租屋處有個前任房客留下來的水泥小陽台，我從去年起一直在想——能不能自己鋪一層木製平台讓它好看一點？也呼應一下這個 blog 的名字（DeckMatt 的 deck 就是甲板/木製平台）。

研究了 3 個月，這篇是我自己 deck building 101 的筆記。寫給跟我一樣想做但還沒動工的人。

為什麼想做

那個水泥陽台真的很醜。也不是說它不能用，只是每次走出去都會有「這是工地不是家」的感覺。鋪上木頭後氣氛應該完全不同——可以擺一張椅子、放杯咖啡、種一兩盆植物。

另外朋友最近在新莊買了房子，他自己搭了一個 deck，週末邀我去坐——我去了之後完全被說服。同樣的空間，鋪上木頭那一刻整個變了。

調研發現的 3 件事

1. 木材選擇比想像複雜

本來以為「就買松木嘛」。結果發現有：

  • 南方松（防腐）：最便宜，但化學處理過，碰水會褪色
  • 紅雪松（cedar）：天然防腐，氣味好但軟，不耐重
  • 柚木：頂級耐用，但 1 坪要 NT$ 10,000+
  • 塑木（WPC）：不會壞但塑膠感重，不像「真的木頭」

我目前傾向紅雪松——預算中等、氣味好、適合台灣濕度。但聽說 5-7 年就要重新上漆。

2. 結構比表面難

原以為「就釘木板上去就好」。實際上要先做：

  • 底部 joist（地龍骨）— 每 30-45 cm 一根
  • 調整水平（陽台地面可能不平）
  • 留水排（雨水流向）
  • 固定方式（用螺絲還是 deck clip）

YouTube 上美國 DIY 影片看起來簡單，但他們的 deck 是花園地面，跟台灣陽台水泥地完全不同。地面性質決定 70% 難度。

3. 工具預算比木材預算還高

我沒有電鑽、沒有圓鋸、沒有水平儀、沒有打釘槍。要做這個 project 工具入手大概：

  • 充電電鑽 + 起子組：NT$ 2,500
  • 圓鋸：NT$ 3,500
  • 水平儀 + 角尺：NT$ 800
  • 砂磨機（最後修邊用）：NT$ 1,800
  • 輔助：手套、護目鏡、口罩：NT$ 600

工具總共 NT$ 9,200。木材如果用紅雪松約 NT$ 8,000。所以第一次做的話工具花得比材料多。但工具未來其他 project 還能用，這樣想就 OK 了。

5 個現實考量（為什麼我可能會放棄）

1. 我是租屋，不是自己家

房東同意嗎？我下個月要去問。如果不行就是 dealbreaker。

2. 拆除時怎麼辦

如果退租要拆，整個 project 等於白做。除非新房客願意接手。

3. 周末時間真的夠嗎

YouTuber 都說「半天就好」。但他們是專業——我估自己第一次至少 2-3 個全天。

4. 颱風天怎麼辦

台灣陽台會淋雨。沒做好排水會積水、長黴、爛掉。這個我還沒想清楚。

5. 修補成本長期

木頭要每 2-3 年重新上保護漆。5-7 年後可能要換板。長期維護不是 0 成本。

下一步

1. 問房東（這週末）
2. 量陽台尺寸 + 拍照
3. 去特力屋實地看木材
4. 找 1-2 個 YouTube 台灣 DIY 影片再看一遍
5. 評估完才下訂工具

如果做完，會寫一篇「Deck Building 102：我做完了」回報結果。如果放棄，也會寫一篇「為什麼我沒做」當作經驗。寫部落格的好處是不管做不做都有素材。
